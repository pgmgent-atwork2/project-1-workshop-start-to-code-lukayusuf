Tutorial

Stap 1: Installatie

Ga naar de officiële website van Visual Studio Code. Op de startpagina zie je meteen een knop om de versie te downloaden die geschikt is voor jouw besturingssysteem (Windows, macOS of Linux). Klik op de knop om het installatiebestand te downloaden. Zodra je dit hebt gedaan kunnen we aan de slag met programmeren!

1.1 Maak een nieuwe folder

Maak een nieuwe folder met de naam puzzle-slider.

1.2 Open VS Code

Open Visual Studio Code en open de nieuwe folder via de navigatie bovenaan.

1.3 Folderstructuur

Maak de bestanden main.css, app.js en index.html. Gebruik de knoppen bovenaan in de verkenner. Zorg dat de bestandsnamen exact kloppen!

Voeg ook een folder toe genaamd assets. Hierin zullen we de foto van de puzzle toevoegen.

1.4 Installatie Live Server

Ga naar extensions en installeer de Live Server-extensie in Visual Studio Code.

1.5 Opstarten van Live Server

Start de Live Server door onderaan op Go Live te klikken. Dit opent een nieuw browservenster met je project.

We beginnen met het ophalen van onze div grid en met het aanmaken van onze correcte array van puzzel tegels.

function tekenBord() {
    grid.innerHTML = '';
}

Met het volgende stukje code lopen we door onze array van tegels en geven we elke tegel twee waarden. 

tegels.forEach((tegelWaarde, huidigePositie) => {

De tegelwaarde is de juiste plek binnen de array om aan de oplossing te komen. De huidige positie is de plek die de tegel op dit moment heeft.

We maken voor elke tegel een div aan binnen het raster van onze puzzle-grid en geven deze een klasse genaamd tegel. 

const div = document.createElement('div');
div.classList.add('tegel');

Om te kunnen schuiven in onze puzzel is er ook altijd een lege tegel. In dit geval stellen we de lege tegel voor met null. Is de tegel null, dan geven we deze de klasse leeg. 

if (tegelWaarde === null) {
    div.classList.add('leeg');
}

Is de tegel niet leeg? Dan moeten we ervoor zorgen dat het juiste deel van de afbeelding getoond wordt.

De x-positie berekenen

const x = (tegelWaarde % 3) * -100;

Hier wordt berekend hoeveel de achtergrond horizontaal moet verschoven worden.

De y-positie berekenen

const y = Math.floor(tegelWaarde / 3) * -100;

Hier wordt berekend hoeveel de achtergrond verticaal moet verschoven worden. 

div.style.backgroundPosition = `${x}px ${y}px`;

Deze regel bepaalt welk deel van de achtergrondafbeelding zichtbaar is binnen één tegel.

Als laatste voegen we een onclick toe aan de tegels. Deze roept de functie probeerSchuiven aan. 

div.onclick = () => probeerSchuiven(huidigePositie);
grid.appendChild(div);

Deze regel zet de gemaakte div effectief in het HTML-element grid.

Stap 5: De puzzel weergeven met CSS

Onze puzzel werkt ondertussen al in JavaScript, maar op de pagina is ze nog niet goed zichtbaar. Daarom gebruiken we CSS om het puzzelbord zichtbaar te maken. 

#puzzle-grid {
    display: grid;
    grid-template-columns: repeat(3, 100px);
    grid-template-rows: repeat(3, 100px);
    gap: 3px;
    padding: 5px;
    background-color: #333;
}

Met display grid maken we van onze container een grid-layout. De puzzel bestaat uit 3 kolommen en 3 rijen. 

.tegel {
    width: 100px;
    height: 100px;
    background-image: url('/assets/puzzle.jpeg');
    background-size: 300px 300px;
    cursor: pointer;
}

Voor deze puzzel gebruiken we best een vierkante afbeelding. 

.leeg {
    background-color: #d1d9e0;
    background-image: none;
    cursor: default;
}

De klasse leeg wordt toegevoegd aan de tegel die geen waarde heeft. 

.tegel:not(.leeg):hover {
    transform: scale(1.03);
    filter: brightness(1.1);
}

Dankzij :not(.leeg) zorgen we ervoor dat het hover-effect niet toegepast wordt op het lege vakje.

Stap 6: De puzzel schudden

Om er een echt spel van te maken, moeten we de tegels willekeurig door elkaar plaatsen.

Plaats dit onder je puzzelbord in index.html:

<button id="schudKnop">Schudden</button>

Nu de knop in onze HTML staat, moeten we deze ook selecteren in JavaScript. 

const schudKnop = document.getElementById("schudKnop");
schudKnop.addEventListener("click", schudPuzzel);

De functie schudPuzzel moeten we nog maken. 

function schudPuzzel() {
    tegels.sort(() => Math.random() - 0.5);
    tekenBord();
    checkWinst();
}

Stap 8: Controleren of de puzzel opgelost is

We voegen een leeg tekstvak toe met het id win-bericht. 

<p id="win-bericht"></p>

Daarna selecteren we dit element in JavaScript:

const berichtElement = document.getElementById("win-bericht");

De functie checkWinst() controleert of de puzzel opnieuw in de juiste volgorde staat. 

function checkWinst() {
    const gewonnen = tegels.every((tegel, index) => {
        if (index === tegels.length - 1) {
            return tegel === null;
        }

        return tegel === index;
    });

    if (gewonnen) {
        berichtElement.textContent = "Proficiat! Je hebt de puzzel opgelost.";
    } else {
        berichtElement.textContent = "";
    }
}

Stap 9: Een oplosknop toevoegen

Voeg in je HTML onder de schudknop deze knop toe: 

<button id="oplosKnop">Oplossen</button>

De knop krijgt de id oplosKnop. Hierdoor kunnen we deze knop later makkelijk selecteren in JavaScript. 

const oplosKnop = document.getElementById("oplosKnop");
oplosKnop.addEventListener("click", resetPuzzel);

De functie resetPuzzel zet de array tegels opnieuw naar de oorspronkelijke, opgeloste volgorde. 

function resetPuzzel() {
    tegels = [0, 1, 2, 3, 4, 5, 6, 7, null];
    berichtElement.textContent = "";
    tekenBord();
    checkWinst();
}

Je hebt nu een volledig werkende schuifpuzzel gebouwd. De tegels kunnen verschoven worden, de puzzel kan geschud worden en er wordt gecontroleerd wanneer de speler gewonnen heeft.
